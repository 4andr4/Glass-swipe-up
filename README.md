# Glass swipe up

Interaktivna grafika v stilu stekla s potegom navzgor in obratno.

# Tehnologija
  - WebGPU + JavaScript
  - .NET Blazor WASM
  - CSS

# Lastnosti
  - zamik premika med miško in grafiko za učinek teže
  - prelomna točka nekje na sredini ekrana, da steklo skoči na svoje mesto gor ali pa dol
  - učinek popačenja ozadja stekla je proceduralen (v ozadju je ena modra črta, ki je vidna samo na steklu za minimalnost)
